<decisions_made>

- GPIO18=temp_down, GPIO19=temp_up (swapped from original assumption -- confirmed via test)
- 0x34="1" in setpoint display mode (VS300FL4 uses different segment pattern during setpoint flash)
- Partial frames (any '?' digit) dropped silently -- transitional states not actionable
- Confidence sensor: publish-on-change only -- 60Hz unconditional publish was unnecessary
- Test buttons use TEST_PRESS phase (bypasses re-home, single raw press, goes to COOLDOWN)
- Restart/SafeMode buttons: entity_category=diagnostic (so they appear in device diagnostic section)
- Protocol is synchronous clock+data (like SPI), NOT RS-485 UART
- Pin 6 = Clock (24 pulses/frame at 60Hz), Pin 5 = Data (sampled on rising edge)
- 0x73 = "9" on VS300FL4 (NOT 0x7B as GS510SZ -- no bottom segment d)
- Re-home strategy: slam 25x to 80F floor, count up to target
- AQY212EH photorelays for galvanic isolation
</decisions_made>
